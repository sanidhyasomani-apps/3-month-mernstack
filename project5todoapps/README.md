# Todo App

A simple and responsive Todo App built using HTML, CSS, and JavaScript.

## Features

* Add new tasks
* Delete tasks
* Save tasks using Local Storage
* Tasks persist after page refresh
* Responsive and clean UI
* Dynamic DOM manipulation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Concepts Practiced

* DOM Access (`getElementById`)
* Event Listeners
* DOM Manipulation (`createElement`, `appendChild`, `remove`)
* Functions
* Arrays (`push`, `filter`, `forEach`)
* Local Storage
* JSON (`parse`, `stringify`)
* State Persistence

## How to Use

1. Enter a task in the input field.
2. Click the **Add** button.
3. The task will be added to the list and saved in Local Storage.
4. Click the **Delete** button to remove a task.
5. Tasks remain available even after refreshing the page.
