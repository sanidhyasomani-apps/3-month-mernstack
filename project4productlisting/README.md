# Product Listing App

A responsive Product Listing application built using HTML, CSS, and JavaScript.

## Features

* Display products dynamically using JavaScript
* Search products by name
* Filter products by category
* Filter products by maximum price
* Sort products by price

  * Low to High
  * High to Low
* Toggle between Grid View and List View
* Responsive design for desktop, tablet, and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

### Search

Users can search products by entering a product name. The product list updates in real time.

### Category Filter

Products can be filtered by category:

* Electronics
* Clothing
* Books
* Sports
* Furniture

### Price Filter

Users can enter a maximum price to display products within their budget.

### Sorting

Products can be sorted by:

* Price: Low to High
* Price: High to Low

### View Toggle

Users can switch between:

* Grid View
* List View
