# Personal Portfolio Website

## Overview

A modern and responsive personal portfolio website developed using HTML, CSS, and JavaScript. The website showcases my profile, technical skills, projects, and contact information while demonstrating responsive design, CSS animations, DOM manipulation, and browser storage concepts.

## Features

* Responsive and modern user interface
* Sticky navigation bar for easy navigation
* About section with profile image and social links
* Skills section built using CSS Grid
* Project showcase cards with hover effects
* Contact form with JavaScript functionality
* Form data storage using Session Storage
* Smooth animations and transitions
* Footer with social media links
* Back-to-top navigation button

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Flexbox
* CSS Grid
* Session Storage API
