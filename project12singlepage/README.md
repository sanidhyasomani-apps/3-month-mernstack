# MyWebsite

A simple multi-page React website built with React Router DOM and Tailwind CSS.

## Features

- Responsive header/navigation bar
- Active navigation links using `NavLink`
- Home page
- About page
- Contact page with a controlled form
- Responsive design using Tailwind CSS
- Clean UI with cards, shadows, buttons, and hover effects

## Tech Stack

- React
- React Router DOM
- Tailwind CSS
- Vite

## Pages

- `/` — Home page
- `/about` — About page
- `/contact` — Contact page
