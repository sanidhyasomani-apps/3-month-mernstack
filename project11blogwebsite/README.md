# BlogSphere - React Blog Application

## Overview

BlogSphere is a simple and responsive blog application built with React, React Router, and Tailwind CSS. It allows users to browse blog posts, view detailed blog content, and learn about the platform through a dedicated About page.

## Features

* Responsive Home Page
* Blog Cards with Hover Effects
* Dynamic Blog Detail Pages using React Router
* About Page with Project Information
* Modern UI with Tailwind CSS
* Client-Side Routing

## Technologies Used

* React.js
* React Router DOM
* Tailwind CSS
* JavaScript (ES6+)
* Vite

## Routes

| Route       | Description                    |
| ----------- | ------------------------------ |
| `/`         | Home Page displaying all blogs |
| `/blog/:id` | Dynamic Blog Details Page      |
| `/about`    | About BlogSphere Page          |

## Concepts Practiced

* Functional Components
* JSX
* Props
* Array Mapping
* React Router (`BrowserRouter`, `Routes`, `Route`, `Link`, `useParams`)
* Conditional Rendering
* Tailwind CSS Styling
* Responsive Design

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
