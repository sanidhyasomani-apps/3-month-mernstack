# Calculator App

A simple Calculator application built using **HTML, CSS, and JavaScript**. The application performs basic arithmetic operations and demonstrates the use of reusable functions, DOM manipulation, switch-case statements, and input validation.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Input validation
* Division by zero handling
* Responsive and clean user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
calculator/
│
├── index.html
├── style.css
└── script.js
```

## How It Works

1. Enter two numbers in the input fields.
2. Select an operation:

   * Add (+)
   * Subtract (-)
   * Multiply (*)
   * Divide (/)
3. The result is displayed instantly on the screen.

## JavaScript Concepts Used

* Functions
* Switch Case
* DOM Manipulation
* Event Handling
* Template Literals
* Input Validation
* Conditional Statements

## Validation Implemented

* Prevents calculations when input fields are empty.
* Handles division by zero gracefully.
* Displays meaningful error messages for invalid operations.
