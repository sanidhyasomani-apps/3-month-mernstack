# Quiz App

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript. The app displays multiple-choice questions, allows users to select answers, tracks their score, and shows the final result at the end of the quiz.

## Features

* Multiple-choice questions
* Dynamic question rendering
* Option selection highlighting
* Score tracking
* Next question navigation
* Final score display
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

1. Questions are stored in an array of objects.
2. The current question and its options are displayed dynamically.
3. The user selects an answer by clicking an option.
4. The application checks whether the selected answer is correct.
5. The score is updated accordingly.
6. The next question is displayed until all questions are completed.
7. The final score is shown at the end of the quiz.
