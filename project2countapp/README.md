# Counter App

A simple and interactive Counter Application built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates core JavaScript concepts such as DOM manipulation, event handling, and closures.

## Features

* Increment the counter value
* Decrement the counter value
* Reset the counter to zero
* Prevents negative values
* Alert when the counter reaches zero
* Responsive and modern UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Concepts Practiced

* Variables and Functions
* DOM Manipulation
* Event Listeners
* Closures
* Conditional Statements
* JavaScript Objects
